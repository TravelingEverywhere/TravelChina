# nodejs安装方法
	
	首先需要到官网下载nodejs的安装包，地址：http://nodejs.org/，根据官网指引，下载相应系统的安装包即可。
    安装完成后，打开命令提示符，安装需要的模块，此脚本对应需要的模块有三个，分别是：
    1. url。
    2. path。
    3. fs。
    安装方法如下：
    "npm install url -g"
    "npm install path -g"
    "npm install fs -g"
    将以上三条命令敲入命令提示符即可，等待其自动下载并安装完成。
    -g为全局标识，否则为局部安装，建议加上-g标识，移动的时候只移动脚本文件就可以了。
